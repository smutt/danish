TEST PAGE

This is a DNSSEC/DANE test page. It uses a certificate from Let's Encrypt with corresponding TLSA records in the DNS. The DNSSEC and DANE configuration should be correct.

To learn more about DNSSEC and DANE visit the Internet Society's Deploy360 Programme page on it.

NIC.CZ offers some great browser extensions for validating DNSSEC and DANE.

NIST offers a web based tool for validating this website's DANE configuration.

Looking for an HTTPS website with a corresponding TLSA record that doesn't match its certficate? Try bad-hash.synonomic.com

If you see a closed lock below, your DNS resolver performs DNSSEC validation.
